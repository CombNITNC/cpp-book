# オブジェクトたち

データのかたまり

---

この章で、多様なデータ型を扱うことができるようになる。

# この章のゴール

- いろいろな型に触れる
- 型につけられる指定子を覚える
- クラスを使いインスタンスを作る
- インスタンスのメンバ関数を呼び出す

所要時間: 80 ~ 90 分くらい
