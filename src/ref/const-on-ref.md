# const の参照

<○><○>

---

今度は `const` と参照を組み合わせてみよう。

```cpp
int value = 0;
int const &ref = value;

// いろいろしてみる
int read = ref; // OK、読み込める
ref = 2; // NG、書き込めない
```

このように、**`const` な変数への参照** という扱いになる。

*他の変数から値を読み取るだけ* の場合に、コピーを防ぐときに使う感じ。
