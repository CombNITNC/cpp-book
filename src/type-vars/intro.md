# 型と変数

ここが情報センターね!

---

この章で、整数や小数をそれなりに取り扱えるようになる。あと入力の基本をやる。

# この章のゴール

* 型の意味を理解する
* 整数と浮動小数点数での演算の違いを把握する
* 整数と浮動小数点数を相互に変換できる
* 型を理解して変数を作る
* 変数に使える名前の仕組みを覚える
* 入力が受け付けられるようになる

所要時間: 80 ~ 90 分くらい
